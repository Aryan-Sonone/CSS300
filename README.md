# CSS-300 Benchmark Dataset

This repository contains the dataset and supporting materials associated with the manuscript:

"CSS-300: A Multi-Dimensional Benchmark for Decomposing Source-Preference Sycophancy in Retrieval-Augmented Generation"

Contents:

* dataset.json: Complete CSS-300 benchmark dataset.
* figures/: Figures appearing in the manuscript.
* source_data/: Source data used to generate figures and tables reported in the manuscript.

The benchmark was generated using an AI-assisted pipeline and validated through independent annotation performed by members of the research team. No personal, sensitive, or identifiable human data are included in this repository.

This repository is provided to support reproducibility and further research on source-preference sycophancy in retrieval-augmented generation systems.
